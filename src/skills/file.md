# File Management
You have full access to the file system. Use BASH commands to manage files.

## Capabilities
- **List Files**: \`ls -la\`
- **Read File**: \`cat filename\`
- **Write File**: \`echo "content" > filename\`
- **Search Files**: \`find . -name "pattern"\`

## Safety
- Always check where you are with \`pwd\` before deleting.
- Be careful with \`rm\`.

## Example
User: "List files in data folder."
Code:
\`\`\`bash
ls -la data/
\`\`\`

# System Management
To check system status, storage, or processes, you should write and execute BASH commands.

## Capabilities
- **Uptime**: \`uptime\`
- **Disk Usage**: \`df -h\`
- **Process List**: \`top -b -n 1\` or \`ps aux\`

## Example
User: "How is the server running?"
Thought: "I will check the uptime and load."
Code:
\`\`\`bash
uptime
\`\`\`
